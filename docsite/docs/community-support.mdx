# Community & Support

This page contains links to community channels and resources for getting help with `rb-sys`.

## Getting Help

- **[Oxidize Ruby Slack](https://join.slack.com/t/oxidize-rb/shared_invite/zt-16zv5tqte-Vi7WfzxCesdo2TqF_RYBCw)**: For questions, sharing projects, and community support.
- **[GitHub Issues](https://github.com/oxidize-rb/rb-sys/issues)**: To report a bug or request a new feature.

## Learning Resources

- **[rb-sys API Docs (docs.rs)](https://docs.rs/rb-sys/)**: API documentation for the `rb-sys` crate.
- **[Magnus Documentation](https://github.com/matsadler/magnus)**: Documentation for the recommended high-level wrapper library.
- **Real-World Examples**:
  - [oxi-test](https://github.com/oxidize-rb/oxi-test) - Canonical example of `rb-sys` usage.
  - [wasmtime-rb](https://github.com/bytecodealliance/wasmtime-rb) - WebAssembly runtime for Ruby.
  - [blake3-ruby](https://github.com/oxidize-rb/blake3-ruby) - BLAKE3 cryptographic hash function.
